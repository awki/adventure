Colossal Cave Adventure
This Colossal Cave Adventure is based on the original 1977 Crowther/Woods PDP-10 Fortran IV code for the classic 350 point game. The Javascript version here, ported by Xenoveritas, is as faithful as possible in both game play and internal structure. This, coupled with termlib, a video display terminal emulator in Javascript, brings an authentic 1977 Crowther/Woods experience to your Web browser.
